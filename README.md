# Cinematic Portfolio | Manthan Railkar

A premium, broadcast-quality cinematic portfolio website inspired by the Formula 1 2026 intro sequence. 

## Features
- **Next.js 14+ (App Router):** High-performance architecture and smooth page rendering.
- **True 3D Environment:** React Three Fiber (R3F) with volumetric lighting, metallic materials, and particle systems.
- **Cinematic Post-Processing:** Bloom, Chromatic Aberration, Noise, and Scanlines.
- **Kinetic Typography:** GSAP-driven masking reveals and horizontal sweeps.
- **Ultra-Fluid Scrolling:** Lenis integrated with GSAP ScrollTrigger for momentum-driven cinematic transitions.
- **Tailwind CSS:** Modern luxury motorsport aesthetic with a deep black and neon red palette.

## Tech Stack
- **Framework:** Next.js
- **Animations:** GSAP, ScrollTrigger, Framer Motion
- **3D Graphics:** Three.js, React Three Fiber, Drei, Postprocessing
- **Styling:** Tailwind CSS
- **Smooth Scroll:** Lenis

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
